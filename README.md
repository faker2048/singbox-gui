# Singbox GUI

一个基于 Tauri + Next.js 开发的 sing-box 图形界面。

## 功能

支持了配置管理和节点切换

### 配置管理
![配置管理](assets/home-page.png)

### 节点切换
![节点切换](assets/node-switch.png)

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务
pnpm tauri dev
```
